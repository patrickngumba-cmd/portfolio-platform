# Personal Project Showcase App

A modern, responsive React Single Page Application (SPA) built for a creative agency to manage and showcase their portfolio projects.

## Features
* **Dynamic Rendering:** View a list of current projects.
* **Add Projects:** Use the form to instantly add new projects to the showcase.
* **Live Search:** Filter projects by title in real-time.
* **Responsive Design:** Clean, card-based layout that works on desktop and mobile.

## Setup Instructions
To run this project locally:

1. Clone the repository to your local machine.
2. Navigate to the project directory: `cd portfolio-platform`
3. Install the dependencies: `npm install`
4. Start the development server: `npm run dev`
5. Open your browser and navigate to the localhost link provided in the terminal.

## Technologies Used
* React (Functional Components, Hooks: `useState`)
* Vite
* CSS3
